Final chrono.js Feature Set
Core Scheduling
.after(duration, fn)

.every(duration, fn, [count])

.start(), .pause(), .resume(), .cancel(), .reset()

Looping
.loop(n) — infinite or fixed number of loops

.onFinish(fn) — runs when loop ends or timeline completes

Conditional Flow
.if(predicate)

.unless(predicate)

.while(predicate).every(...)

.doWhile(predicate).every(...)

Branching
.label(name)

.jumpTo(name)

New Utility Modules
debounce(duration, fn)

throttle(duration, fn)

waitFor(condition, options)

retry(fn, { attempts, delay, backoff })
